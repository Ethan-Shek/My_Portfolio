Project Overview

Project Overview

CrunchTime: Special Anime Edition is a passion project designed to celebrate anime culture through engaging, user-centered design. My goal was to create a magazine that captures the excitement and color of anime while maintaining clarity and visual balance, similar to how good UX/UI design merges creativity with usability

Design Approach

Design Approach

I treated each spread like an interface every element needed to guide the reader’s attention intuitively.
✦ Visual Hierarchy: Bold headlines and sectional breaks make dense content feel approachable, similar to navigation design in web layouts.
✦ Color & Composition: I used bright, energetic hues inspired by anime posters, balanced with white space to avoid visual fatigue.
✦ Consistency: Repeated motifs (banners, rounded shapes, footer waves) provide flow and unity across pages.
✦ Micro-Interactions (Print Equivalent): Layered “Underrated” stamps, iconography, and character cameos act like micro-moments of delight—details that reward attentive readers.

Highlights

Highlights

✦ Kaiju No. 8 Feature: Focused on character-driven storytelling. Designed character cards with layered typography to mimic modern web component grids.
✦ Hunter × Hunter Spread: Integrated symbols and props tied to the show’s lore for contextual depth and visual language consistency.
✦ Studio Ghibli Page: Crafted a warm, home-like aesthetic using textures and layouts inspired by bookshelf UI metaphors, evoking comfort and nostalgia.
